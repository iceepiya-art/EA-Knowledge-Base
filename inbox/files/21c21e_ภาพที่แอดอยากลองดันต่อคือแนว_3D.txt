กำลังลองทำ Multi-Agent ให้มี “ออฟฟิศ 3D” ของตัวเอง
ช่วงนี้แอดกำลังศึกษา Pixel Agents อยู่ครับ เป็น repo ที่ทำให้ AI Agent ไม่ได้อยู่แค่ใน Terminal หรือ log ยาว ๆ แต่กลายเป็นตัวละครในออฟฟิศเล็ก ๆ ที่เรามองเห็นได้ว่า Agent แต่ละตัวกำลังทำอะไรอยู่
ไอเดียนี้น่าสนใจมาก เพราะปัญหาของ Multi-Agent ไม่ใช่แค่ “สั่งให้หลายตัวทำงานพร้อมกัน” แต่คือเราต้องรู้ให้ทันว่าแต่ละตัวกำลังทำอะไรอยู่
ไม่งั้นจาก Multi-Agent จะกลายเป็น Multi-งง อย่างรวดเร็ว 555
ภาพที่แอดอยากลองดันต่อคือแนว 3D / isometric cartoon office ประมาณนี้
Agent แต่ละตัวมีโต๊ะ มีหน้าที่ มีสถานะของตัวเอง เช่น
ตัวหนึ่งกำลังเขียนโค้ด
ตัวหนึ่งกำลังอ่านไฟล์
ตัวหนึ่งกำลังรอ permission
อีกตัวเป็น sub-agent ที่ถูกแยกไปทำงานเฉพาะทาง
ถ้างานเสร็จหรือมีอะไรต้องกดอนุมัติ ก็แสดงเป็น animation หรือ bubble ให้เห็นชัด ๆ
สิ่งที่ทำให้โปรเจกต์นี้น่าสนใจ ไม่ใช่แค่ความน่ารักของตัวละคร แต่คือมันช่วยเปลี่ยน Multi-Agent จากระบบหลังบ้านที่ดูยาก ให้กลายเป็น workspace ที่คนใช้งานมองภาพรวมได้ทันที
เพราะเวลา Agent หลายตัวทำงานพร้อมกัน การดูแค่ Terminal บางทีก็เหมือนนั่งดูหม้อข้าวเดือด
มีเสียง มี movement แต่ไม่รู้ข้างในสุกหรือยัง 555
แนวที่แอดกำลังคิดคือไม่รีบรื้อระบบให้เป็น 3D เต็มรูปแบบก่อน แต่จะเริ่มจากทางที่ปลอดภัยกว่า คือทำให้ asset และฉากเดิมมีฟีล 3D cartoon มากขึ้น โดยยังใช้ logic เดิมของ Pixel Agents ให้มากที่สุด
ประมาณนี้
1. ใช้ระบบ Multi-Agent เดิม
2. เปลี่ยนภาพออฟฟิศจาก pixel 2D ให้ดูเป็น 3D cartoon
3. ทำฉาก workspace แบบ rooftop / office / lab
4. เพิ่ม furniture และ character asset ให้ดูมีมิติ
5. ให้สถานะของ Agent สื่อสารผ่านภาพ ไม่ใช่ต้องอ่าน log อย่างเดียว
ถ้าออกมาเวิร์ก แอดจะเอามาแจกให้ลองกันครับ
ส่วนตัวมองว่าแนวนี้มีประโยชน์มากสำหรับคนที่เริ่มเล่น Multi-Agent เพราะมันทำให้เราเข้าใจระบบง่ายขึ้น เห็นภาพชัดขึ้น และน่าจะสนุกกว่าเปิด Terminal หลายหน้าต่างแล้วถามตัวเองว่า “ตัวไหนทำอะไรอยู่แล้วนะ”
